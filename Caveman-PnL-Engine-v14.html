Note on the per-kW model: electrical is largely fixed (one DC isolator, one breaker, one label kit regardless of size) plus a small per-kW component (cable). Pure per-kW undercharges small jobs (1–4 panels) and overcharges large ones (28+ panels). For ±20% sizing, per-kW is fine. For one-offs or very large jobs, use the BoM tables below directly.

Site adjustments & compliance

Baseline assumes: JA Solar 475W panels · standard tin/Colorbond, 15–30° pitch · single roof face · switchboard within 5m · single-storey access.

Cost adders for non-standard sites

  • Tile roof: +$15–25/panel for tile hooks vs L-feet
  • Two-storey: +$200–400 for scaffolding/edge protection
  • Multiple roof faces: +$50–100 per face for split rails/extra cable
  • Steep pitch (>30°): +$300–500 for height-safe rigging
  • Spreader plates required: +$3–5 per L-foot (commercial tin)
  • Trenching: $20–30/m for outdoor runs
  • Switchboard upgrade: $1,500 (1ph) / $2,000 (3ph) basic; $2,500–3,500 if relocation needed

Compliance — AS/NZS standards

  • AS/NZS 5033 — PV array installation (DC isolator placement, signage, string sizing)
  • AS/NZS 4777 — Grid-connected inverter installation
  • AS/NZS 3000 — Wiring Rules (general)
  • AS/NZS IEC 60947-3 — Isolator switch (DC-PV2 mark required since May 2022)
  • Rooftop DC isolators: IP66, UV-stabilised, current compliance mark
  • WEEB / earthing clip at every panel-to-rail interface (bonded grounding path)

❓ Quick Start Caveman Energy · P&L Engine v14

What this is: A self-contained sales engine that runs in one HTML file. Pin to phone, tablet or desktop. Pricing is driven from the editable Price Table and rules below — change a cost once, all cards / grids / quotes re-price instantly. Per-user state via ?user=YourName in the URL.

G'day - quick rundown of the tabs up top. Tap a tab to open or close it. Here's what each one does:

📊 P&L Calc
Build a job line by line - panels, inverter, battery, extras. Pick the brand and sizes from the dropdowns, set the quantities with the +/− arrows. Punch in a target $ rating and it works backwards to a sale price, or type a sale price and watch the profit. Mini calculator down the bottom if you need to tot something up.
📝 Quote
Turns the job into a clean quoting sheet. Pick the panel and inverter, set quantities, and the labour / sundries / racking lines fill in off the rates. Edit any line if a job's different - change the rate or type straight over the total. Shows retail by default.
🔧 Add-on
For adding to a system that's already there - an extra battery module, an EV charger, a couple of panels. Pick the item, set the quantity, add your labour hours. Prices the add-on on its own (just the gear + labour), not like a full new install.
💲 Price Table
The master list of what everything costs. This is the source of truth - change a price here and it flows through to every tab. Have a look but go careful editing, it changes the numbers everywhere.
🔩 Materials
Reference for the bits and pieces on a job - cabling, racking, mounting, sundries. Quick lookup so you know what goes into a build and roughly what it runs.

Your settings save against your own login, so when you come back it's where you left it. Any dramas, give Steve a buzz.

👤 Three-tier access

URL determines role:

  • ?user=Brooko → master — sees everything, manages all companies + users in the User Access section.
  • ?user=Taylor (once Taylor is added as a company) → company admin — sees Taylor's allowed features + manages Taylor's employees.
  • ?user=Taylor-Luke → employee — sees only the intersection of Taylor's permissions and Luke's per-user overrides.
  • ?user=anything-else → guest — sees everything (backward-compatible).

Feature catalog includes top sections, Brand Builder, sell columns, Map Finance Mode, defaults bar, and per-brand visibility. Block a feature for a company → all their employees lose access; block on one employee → only they lose it.

💾 Defaults bar

  • 💾 Save defaults — snapshots current state (toggles, brand picks, EV, families, target overrides, step overrides, Price Table, etc.) into localStorage under your ?user= key.
  • 📥 Load saved — restores from the snapshot.
  • ↺ Factory — wipes the in-page state back to baked-in defaults, but keeps your saved snapshot available for Load.
  • 🗑️ Wipe — deletes the saved snapshot entirely.
